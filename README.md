# Link Redirect Manager 🔗

一个支持 IP 固定分配、地区限制和访问记录的链接重定向管理系统。

## 功能特性

✅ **IP 固定分配** - 每个 IP 被随机分配到一个固定的子链接，永久锁定  
✅ **地区限制** - 支持按国家代码阻止访问  
✅ **多域名管理** - 创建和管理多个独立的域名  
✅ **访问记录** - 完整的 IP 分配和访问历史  
✅ **Web 管理面板** - 直观的管理界面  

## 快速开始

### 本地开发

\`\`\`bash
bun install
bun run dev
\`\`\`

访问 http://localhost:3000

### 部署到 Railway

1. 连接此 GitHub 仓库到 Railway
2. 添加 PostgreSQL 数据库
3. 自动部署完成！

## 技术栈

- Bun + TypeScript
- PostgreSQL
- 原生 HTTP API
